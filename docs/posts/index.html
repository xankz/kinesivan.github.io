ATEC Concentration Application F21

Current Projects QuestRPG is a VR single-player RPG developed solely by me in Godot for Oculus Quest headsets. This project was meant to explore and test what an RPG experience would be like in VR, for when I later plan on developing an open-world multiplayer version. Working in a relatively-new open-source engine proved to be difficult for this project, however, I gained a lot of experience working in 3D and VR along the way.
